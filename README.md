# 國寫批改助手 📝

專業的國語文寫作自動批改與教學助手系統，支援學測國寫與國中會考作文評分。

## 功能特色

- ✅ **學測國寫批改** - 支援知性題與情意題
- ✅ **會考作文批改** - 六級分制評分
- ✅ **四大面向評分** - 立意取材、組織結構、遣詞造句、標點錯字
- ✅ **詳細評分規準** - 依據大考中心官方標準
- ✅ **響應式設計** - 支援手機、平板、電腦

## 評分標準

### 學測國寫
| 題型 | 配分 | 評分等級 |
|------|------|---------|
| 知性題（第一大題） | 25 分 | A+, A, B+, B, C+, C |
| 情意題（第二大題） | 25 分 | A+, A, B+, B, C+, C |

### 會考作文
| 級分 | 表現描述 |
|------|---------|
| 6 級分 | 優秀 |
| 5 級分 | 一般水準之上 |
| 4 級分 | 一般水準 |
| 3 級分 | 表達不充分 |
| 2 級分 | 嚴重問題 |
| 1 級分 | 極嚴重問題 |
| 0 級分 | 離題/空白 |

## 技術架構

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **字型**: Noto Sans TC (Google Fonts)
- **部署**: GitHub Pages

## 本地開發

```bash
# Clone 專案
git clone https://github.com/YOUR_USERNAME/essay-grading-assistant.git

# 進入專案目錄
cd essay-grading-assistant

# 使用任意 HTTP server 開啟
# 例如 Python
python -m http.server 8000

# 或使用 VS Code Live Server 擴充功能
```
